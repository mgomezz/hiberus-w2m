# Hiberus-W2M

# Run with Docker
Open cmd and run the following commands:

  1. cd hiberus-w2m
  2. docker-compose up
  3. Server will launch in http://localhost:3080 and UI in http://localhost:4200

# Run Without Docker Compose

# API
Open cmd and run the following commands:
  1. cd api
  2. npm install
  3. npm run dev
  4. the server will launch in http://localhost:3080

# UI
Open cmd and run the following commands:
  1. cd ui
  2. npm install
  3. npm start
